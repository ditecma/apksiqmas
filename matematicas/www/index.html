Obtención de una expresión cuadrática para definir el enésimo término de una sucesión

En estas sesiones aprenderemos a encontrar la expresión general cuadrática de la forma Y = X2, que nos ayude a calcular el enésimo término de sucesión numérica o figurativa.

Para comenzar definiremos algunos elementos que ocuparemos durante estas sesiones.

  • Sucesión numérica o figurativa. Es una serie de elementos numéricos o figuras geométricas ordenadas. Algunos ejemplos son:

Sucesión numérica

Todos los números que forman esta sucesión y que están separados por las comas son llamados elementos (1, 4, 9, 16...)

Sucesión figurativa

Todas las figuras que forman esta sucesión y que están separadas son llamadas elementos.

  • Enésimo término. Dentro de una sucesión numérica o figurativa como las anteriores, el enésimo término es el que se identifica con un subíndice  y se refiere a cualquiera de los elementos que forman a la sucesión al depender del valor que se la asigna a .

Ejemplo:

En la siguiente sucesión numérica encuentra el valor de n = 4

25, 36, 49, 64, 81, 100

El problema nos pide encontrar n = 4, esto quiere decir que encontremos el cuarto elemento de esta sucesión. Entonces:

25, 36, 49, 64, 81, 100

Como lo muestra este ejercicio el cuarto elemento es el 64.
